# islamic
A library for all Islamic data.
The library contains multiple files for:
  - All god names in Arabic, English, Persian, Russian and Kurdish
  - All Azkar in Arabic, English, Persian, Russian and Kurdish
  - Islamic radio channels in multiple languages
  - All supported countries for prayer locations
  - All calculation methods for prayer
  - Tafsir Quran in Arabic for Jalalayn, Ibn Katheer, Qurtubi and Muyassar
